Sentence Comprehension as a Cognitive Process: A Computational Approach

Shravan Vasishth and Felix Engelmann
This web page accompanies a book to be published by Cambridge University Press. The web page provides all the code that lies behind the modeling presented in the book. This project was funded by the Volkswagen Foundation (grant 89 953). The github repository behind this page is here.


Instructions for reproducing analyses:
  1. Clone the repository (if you know how to use github), or download the master directory as a zip archive (if you are unfamiliar with github)
  2. Download the data.zip directory and place it in the main directory of this repository. Without these data the R code that comes with the book will not work.
